Formulário Renda Fixa
Use um valor aproximado que você pretende investir.
Informe o horizonte de tempo desejado (ex.: 360, 720).
• Conservador: foco em segurança.
• Moderado: equilíbrio entre risco e retorno.
• Turbo: prioriza retorno, aceitando mais risco.

Este é um protótipo. Os dados atuais são apenas exemplos mockados para testar a lógica de ranking.
